# Plan - Fundamente Java pentru începători

Conținut educațional în română, destinat elevilor fără experiență în Java.
Fiecare lecție este un fișier `.md` în `docs/java/` și predă prin exemple concrete.

---

## Lecții

- [x] **L1** - `docs/java/01-primul-program.md` — Hello World, structura unui program Java, `System.out.println()` vs `System.out.print()`
- [x] **L2** - `docs/java/02-variabile.md` — Tipuri de date (`int`, `double`, `String`, `boolean`, `char`), declarare și inițializare, folosirea variabilelor în output
- [x] **L3** - `docs/java/03-citire-consola.md` — Clasa `Scanner`, citirea de `int`, `double`, `String` de la tastatură
- [x] **L4** - `docs/java/04-operatii-aritmetice.md` — Operatori `+`, `-`, `*`, `/`, `%`, împărțire întreagă vs reală, ordinea operațiilor
- [x] **L5** - `docs/java/05-if-else.md` — `if`, `if-else`, `if-else if-else`, operatori de comparație și logici (`&&`, `||`, `!`)

---

## Format lecții

Fiecare fișier va conține:
1. Titlu și scurtă descriere a conceptului
2. Structura minimă necesară (boilerplate explicat)
3. Exemple progresive (simplu → complex)
4. Observații (`> **Obs:**`) pentru capcane frecvente
