# Împărțirea cu numere "cu virgulă"

> **Obs:** Cand evaluez a/b, dacă cel puțin unul dintre termeni este “cu virgula” (de tip float sau double), atunci se va calcula impartirea exacta. În caz contrar, se calculeaza catul împărțirii.

> **Obs:** Cand evaluez a%b, dacă cel puțin unul dintre termeni este “cu virgula” (de tip float sau double), atunci obținem EROARE DE COMPILARE. În caz contrar, se calculează restul împărțirii.
